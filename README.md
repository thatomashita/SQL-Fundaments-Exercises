# SQL Fundamentals

## Overview

This repository contains my SQL Fundamentals learning exercises, and practice queries. It demonstrates my understanding of SQL concepts used for data analysis and database management.

## Skills Covered

* SQL Syntax
* SELECT Statements
* Filtering Data with WHERE
* Sorting Data using ORDER BY
* Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)
* GROUP BY and HAVING
* LIMIT Statement

## Project Objectives

The purpose of this project is to:

* Build a strong foundation in SQL.
* Practice querying and analyzing data.


## Tools Used

* Notepad
* Databricks SQL
